# Avatar Storage for Story Weave

This folder contains avatar images for the Story Weave multiplayer game.

## Avatar Requirements

Upload 20 avatar images with the following specifications:

### File Naming
- `avatar_1.webp` through `avatar_20.webp`
- Must use `.webp` format
- Sequential numbering from 1 to 20

### Image Specifications
- Format: WebP
- Recommended size: 128x128 pixels (or any square aspect ratio)
- Should be optimized for web delivery
- File size: Keep under 50KB per image for fast loading

### Style Guidelines
- Use distinct, easily recognizable designs
- Consider diversity in appearance
- Ensure good visibility on both light and dark backgrounds
- Avoid copyrighted characters or imagery

## Usage

Players will select from these 20 avatars during the character setup phase of Story Weave games. The selected avatar will be displayed:
- In the player list during lobby and gameplay
- Next to each story submission
- In the exported Markdown story file

## Example Avatar Ideas

1. Abstract geometric shapes (circles, triangles, squares) in different colors
2. Simple emoji-style faces with different expressions
3. Minimalist animal silhouettes
4. Color-coded badges or shields
5. Nature elements (leaves, stars, flowers)
6. Fantasy-themed icons (wands, crystals, crowns)
7. Sci-fi elements (planets, rockets, robots)
8. Mystery/horror symbols (magnifying glass, masks, keys)
